# Install from go
```sh
go install github.com/ikasoba/saka@latest
```

# Releases

https://github.com/ikasoba/saka/releases

# Usage

```
Usage: ./saka <filename>
  Open file

Usage: ./saka (-e|--show-environment)
  Show environment variables

Usage: ./saka --install-grammar
  Install grammar from config
```
